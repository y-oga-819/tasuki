# ドキュメントビューア

Tasukiでは、Diffの横にドキュメントを並べて表示できます。実装が仕様に沿っているかの確認に便利です。

## Splitモード

**Split**表示モードでは、画面が2つのペインに分割されます：

- **左**: Diffビューア
- **右**: ドキュメントペイン

サイドバーの**Documents**セクションからドキュメントをクリックすると、右ペインに表示されます。

## Viewerモード

ツールバーの**Viewer**タブをクリックすると、ドキュメント重視のレイアウトに切り替わります：

- **左**: 全幅のMarkdownビューア
- **右**: ターミナルペイン

このモードではdiff関連のコントロールとChanged Filesセクションが非表示になり、ドキュメント閲覧に集中できます。

## 対応コンテンツ

### Markdown

GitHub Flavored Markdownの完全なレンダリング：

- 見出し、リスト、引用
- テーブル
- シンタックスハイライト付きコードブロック（Shiki）
- タスクリスト
- リンクと画像

### Mermaid図

Mermaid図はインタラクティブなSVGとしてレンダリングされます：

- ホバーで**ズームボタン**が表示される
- クリックするとフルスクリーンモーダルが開く
- **+/−** ボタンまたはマウスホイールで拡大/縮小
- ドラッグでパン（移動）
- パーセンテージ表示をクリックでズームリセット
- `Escape`キーまたは外側クリックで閉じる

### 設計書

`~/.claude/designs/{repo-name}/` からの設計書がサイドバーの**Design Docs**セクションに表示されます。読み取り専用で、レビュー中のアーキテクチャ決定の参照に役立ちます。

## 外部フォルダ

Tauriデスクトップアプリでは、外部フォルダを追加できます：

1. サイドバーの **Add Folder** をクリック
2. `.md`ファイルを含むフォルダを選択
3. フォルダが新しいサイドバーセクションとして表示される
4. ファイルをクリックしてプレビュー
5. **✕** でフォルダを除去

複数のフォルダを独立して追加可能です。

## レイアウト

- ペイン間の境界線は**ドラッグ可能** — ドラッグしてサイズ変更
- Splitモードではドキュメントペインの幅が**最大900px**に制限（diffが主役）
- Viewerモードではターミナルペインの幅が制限（ドキュメントが主役）
